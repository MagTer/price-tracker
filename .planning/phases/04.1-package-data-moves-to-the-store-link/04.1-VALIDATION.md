---

## Sampling Rate

- **After every task commit:** `poetry run pytest -x -q`
- **After every plan wave:** `poetry run pytest` + `ruff check src tests`
- **Phase gate (D-15 reset is PART of the gate, so it cannot be forgotten):**
  ```bash
  docker compose down -v && docker compose up -d postgres \
    && alembic upgrade head && alembic check && poetry run pytest
  ```
- **Max feedback latency:** ~5 s for unit; ~30 s for the full gate
