<objective>
Rewire every admin READ path onto the computed unit price, and close the third scrape write-path.

Purpose: MODEL-03 says EVERY unit-price consumer recomputes. The admin API has five of them (product list,
product detail, price history, manual check, export), and each currently reads the dropped
`PricePoint.unit_price_sek` column. MODEL-05's autofill/flag rule must also reach `POST /check/{id}` — which
duplicates the service's fetch/parse/record logic inline instead of calling it, and would otherwise record a
price while never filling the quantity, so "Check now" would never do what AC#4 promises.

Output: link row dicts carrying the full package + kr/unit picture; a `GET /products/{id}/links` endpoint for
the product page; `POST /check/{id}` honouring D-07; the export/import shape moved onto the link.
</objective>
